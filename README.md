# Simple Inventory Management System 📦
A console-based Java application designed to manage warehouse inventory efficiently. This project demonstrates core **Object-Oriented Programming (OOP)** principles and basic data structure management.

## 🚀 Features
* **Add Products:** Insert new items with unique IDs, names, and price.
* **Stock Management:** Update product quantities easily.
* **Search & View:** Search for specific items by ID or view the entire inventory list.
* **Delete Items:** Remove products from the system.
* **Input Validation:** Robust error handling to ensure IDs and quantities are entered correctly as numbers.
  
## 📁 Project Structure
* `Product.java`: Defines the product attributes and properties.
* `Inventory.java`: Contains the logic for managing the list of products.
* `Inventory_System.java`: The main entry point that handles user interaction via the console.

## 💻 How to Run
1. Clone the repository.
2. Compile the files: `javac Inventory_System.java`.
3. Run the application: `java Inventory_System`.
